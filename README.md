# SwiftTube

Simple SwiftUI Youtube client with Liquid Glass.

## Features
- Add Channels to see their videos
- Watch Regular views and shorts
- Sync watch history and watch later playlists over iCloud

## Requirements
- Youtube Data API Key
- iOS/iPadOS/macOS 26+
